🧪 Laboratory Update in Magic Alchemy

Greetings, alchemists! ⚗️

Improved tools are now available in the game, which can be purchased from Corpa! Additionally, the laboratory now features an inventory system, making tool management easy and intuitive.

Improved tools can be set up on the table just like basic tools, but their main advantage is increased durability:
* **Basic tool:** lasts for 1 brew, requires 2 spare parts for repair.
* **Improved tool:** lasts for 3 brews, requires only 1 spare part for repair.

💰 Purchase improved inventory using POTION. You can already get it on the pre-market. Note: any quantity of the same type of improved booster can be purchased.

In the future, repairing broken tools will likely be more cost-effective than replacing them, so plan your strategy ahead. Improved tools will also eventually reduce brewing time.

This update is a major step toward turning the laboratory into a full system for managing resources and efficiency. For now, tool durability will not decrease, so you can start upgrading your workspace with these new tools!

The laboratory is becoming deeper. Prepare to plan your brews and build your economy. 🧪
